# MES 笔记

## MES的定义以及标准
*　AMR
```text
AMR(Advanced Manufacturing Research) 将MES定义为位于上层计划管理系统与底层工业控制系统之间的，面向车间的管理系统，可以为操作人员，管理人员
提供计划、执行、跟踪及所有资源(人、设备、物料、客户需求等方面)的当前状态信息。


```
* MESA
```text
MESA(Manufacturing Execution System Association)将MES定义为:能通过信息传递，对从订单下达到产品完成的整个生产过程进行优化管理。当工厂里面
有实时事件发生时，MES能够对此做反应，并用当前的准确数据堆对他们进行指导个处理。这种对状态变化的迅速响应使得MES能够减少企业内部没有附加值的活动，
有效地指导工厂地生产运作过程，从而既能够提高工厂即时交货地能力、改善物料的流通性能，又能提高生产回报率。MES还能通过双向的直接的通信在企业内部和
整个产品供应链种提高有关产品行为的关键任务信息。

除了以上的定义之外，MESA还提出了MES应该具体一下标准的模块：
1、详细生产计划(详细产品计划及排产)：生产顺序和工时平衡需要考虑单个生产线或者整个生产线的产能和资源的利用率。
2、资源管理(资源的位置和状态):管理和监控各种资源的状态，如设备、工具等。
3、文档管理(包括文档控制):管理和正确传递产品、流程、设计和订单的信息，甚至包括现场质量控制的作业指导书等。
4、物料管理(物料管理及配送)：管理现场生产需要的物料及半成品的现场配送，保证产品质量。
5、绩效管理:对比和评价标准数据和现场实际生产的装配、测量数据之间的差异，以保证产品的质量。
6、秩序管理(工作管理)：控制现场操作，并将最新的信息发送给指挥中心或者现场工人。
7、维护管理:计划和实施设备的维护任务，减少由设备故障和宕机引起的生产损失。
8、生产过程管理：控制和管理生产流程，按照既定的流程组织生产。
9、质量管理：记录、跟踪和分析产品生产过程中的各种问题，确保出厂产品的质量达标。
10、数据采集：实时采集现场人工操作、设备运行、物料配送等信息，并进行数据可视化分析。
11、产品跟踪和追溯:跟踪、记录与产品相关的信息，对零件、过程及事件进行记录并保存。

```
